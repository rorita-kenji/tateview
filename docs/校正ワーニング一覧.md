## 詳細ルール

### 字下げ系（行単位）

- **見出し行**（`heading.js` の章/話マーカー）は `indent-missing` / `dialog-indent` 対象外。
- **空白のみの行**（`/^[　\s]*$/`）も字下げ系対象外。
- 行頭が半角 SP / TAB のときは `indent-missing` を出さない（`halfwidth-space` / `tab` / `trailing-space` 側を先に直す想定）。
- `indent-missing` のラベルは `字下げ漏れ　{行頭〜10コードポイント…}`。ハイライト範囲もその頭部分。

### 空白系の優先（三重検出防止）

1. **`trailing-space`** … 行末 `[ \t　]+`
2. **`halfwidth-space`** … 行中の **純粋な半角 SP ランのみ**（TAB のみランはここではなく tab／trailing）
3. **`halfwidth`** … 非空白半角を含むラン。ラン末尾が trailing に重なる部分は end を詰める  
   ラン内の半角スペースは halfwidth に含め、別途 halfwidth-space は出さない

### 半角ランの文字集合

```
[\x20-\x7e｡-ﾟ\uff61-\uff9f]
```

- ASCII 印字＋SP、半角カナ一式
- **除外位置**: ルビ記法範囲内、見出しプレフィックス範囲内（`excluded`）

### 全角英字・全角数字・絵文字（info）

いずれも severity **info**（warning より下の確認用）。連続はラン単位で1件。半角と同様に **ルビ記法範囲・見出しプレフィックス** は `excluded` で除外。

| code | 対象 | 備考 |
|------|------|------|
| `fullwidth-alpha` | `Ａ-Ｚａ-ｚ`（U+FF21–FF3A / U+FF41–FF5A） | 半角 `A-Za-z` は対象外（`halfwidth` 側） |
| `fullwidth-digit` | `０-９`（U+FF10–FF19） | 半角数字は `halfwidth` |
| `emoji` | 下記シーケンス | `※` `★` など通常記号は対象外 |

**絵文字シーケンス**（`scanEmojiRuns`）:

- `\p{Extended_Pictographic}` ＋ 任意 VS16（U+FE0F）＋ 任意肌色（U+1F3FB–1F3FF）＋ 任意タグ列（U+E0020–E007E…U+E007F）＋ ZWJ（U+200D）連鎖
- 国旗: `\p{Regional_Indicator}` × 2
- キーキャップ: `[0-9#*]` ＋ 任意 VS16 ＋ U+20E3
- **隣接する**シーケンスは1ランに結合（間に他文字があれば別件）

### ルビ範囲の収集（除外・ruby-off 用）

- `｜…《…》` : `/｜[^《\n]*《[^》\n]*》/g`
- 漢字連続＋`《…》` : `/[㐀-䶿一-鿿々]+《[^》\n]*》/g`

### ルビ構文エラー

| 条件 | 正規表現イメージ |
|------|------------------|
| 空ルビ `《》` | `/《》/g` |
| 閉じ `》` なし／改行で途切れる | `/《[^》\n]*(?:\n|$)/g`（開始 `《` 位置） |
| `｜` のあと同一行に `《` が続かない | `/｜(?![^《\n]*《)/g` |

### 括弧バランス

対象ペア（`《》` はルビ専用のため **対象外**。ルビ範囲内の括弧もスキップ）:

| 開き | 閉じ |
|------|------|
| 「 | 」 |
| 『 | 』 |
| （ | ） |
| 【 | 】 |
| 〔 | 〕 |
| ［ | ］ |
| 〈 | 〉 |
| ( | ) |
| [ | ] |
| { | } |

- **文書全体**の1スタック（行ごとではない）。意図的な未閉じのあとに別の孤立閉じがあるとペアになり消える点に注意。
- 閉じ不一致: `unmatched-close-bracket`。スタックに別種の開きがあるとき追加で `mixed-bracket`（開き位置、ラベル `開き→実際の閉じ`）。
- 終了時スタック残り: 各 `unclosed-bracket`。

### 連続空行

`\n` + 空白のみ行 + `\n` + 空白のみ行 + `\n` にマッチ（最低でも空行が「3行分」つながるイメージ）。空白は半角SP/TAB/全角SP。

### ソート順（同一 start 付近）

狭い範囲優先のあと、概ね次の code 順（`warnCodeOrder`）:

`ruby-syntax-error` → `unmatched-close-bracket` → `unclosed-bracket` → `tab` → `halfwidth-space` → `halfwidth` → `fullwidth-alpha` → `fullwidth-digit` → `emoji` → `trailing-space` → `period-before-bracket` → `no-space-after-bang` → `odd-leader-dash` → `repeated-chouon` → `double-punctuation` → `lonely-katakana-ni` → `mixed-bracket` → `dialog-indent` → `indent-missing` → `fullspace-only-line` → `consecutive-blank` → `ruby-off`

## UI 側の付帯（検出本体外）

- 一覧表示: `p{ページ} {行}行　{label}`（行＝そのページ内の列 index＋1）
- `indent-missing` は2行表示（1行目ラベル、2行目ヘッド断片）
- 前/次は端でループしない
